Inferno A standalone library for embedding Chrome Devtools CPU flame charts in any application

Features

  • Use mouse to pan/zoom
  • Responsive to window/container size
  • Drag to select time region
  • Attach chart to a custom container of any size

Planned features

  • Multiple charts per page
  • Add your own meta data to the hover display
